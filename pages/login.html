Como funciona o login / entrar no app?

Fazer o login / entrar no app permite que você utilize as informações salvas do seu cadastro para as funcionalidades do sistema. Seus dados são mantidos seguros e somem do seu dispositivo após você realizar logout / sair do sistema.

Meus dados são divulgados?

Os dados do seu cronograma são usados de forma anônima para fins logísticos. Seus demais dados só são disponíveis de forma pública se você desejar. Você pode configurar isso no seu perfil.

Posso usar o aplicativo sem login / entrar no sistema?

Sim, porém você terá restrições no seu uso.
